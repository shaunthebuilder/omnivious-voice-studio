# Omnivious Voice Studio Fun Edition

This is the lighter, non-technical explanation of the product. For the formal product doc, use [omnivious-prd.md](omnivious-prd.md).

## What It Is

Think of Omnivious Voice Studio as a voice acting studio on your laptop.

You bring:
- a sample voice
- some text
- a mood

The app gives you:
- a reusable voice persona
- expressive audio output
- a local library of generated clips

Basically: same voice, different vibes.

## What It Does

Omnivious Voice Studio does four big things:
1. Learns a voice from source audio.
2. Prepares that voice as a reusable persona.
3. Lets you render different styles from that persona.
4. Keeps quality checks and output history visible.

## Why It Feels Different

It is designed to be:
- local first
- open source
- zero recurring API cost
- expressive, not sterile
- transparent about quality

## What The Experience Feels Like

### Step 1: Train A Persona
- give it a file or YouTube link
- let it normalize the source
- wait for certification

### Step 2: Pick A Style
- natural
- news
- drama
- sad
- happy
- charming

### Step 3: Generate
- paste text
- choose preview or final mode
- adjust speed
- render locally

### Step 4: Review
- play the latest result
- browse earlier renders
- keep the ones you like
- delete the ones you do not

## Why It Matters

The product is useful because it combines:
- repeatability
- local control
- creative flexibility
- no per-render token billing

It is less like a one-off AI demo and more like a reusable local workstation.

## Responsible Use

This is still a powerful AI tool. It should be used responsibly, legally, and with appropriate consent.

Do not use it for:
- deception
- impersonation
- fraud
- abusive or misleading synthetic media
